# Melange37_ERP

Jewelry Ecommerce ERP 애플리케이션의 기본 프로토타입입니다.

## 실행 방법
별도 빌드 없이 정적 파일로 동작합니다.

```bash
python3 -m http.server 8000
```

브라우저에서 `http://localhost:8000` 접속 후 사용하세요.

## 제공 기능
- 사이드바 탭 기반 테이블 이동
- 재료 주문 테이블: 개당 원가/총비용 계산
- 패키지 비용 테이블: 상품 종류별 패키지 합계
- 상품 제작 테이블: 재료+패키지 기반 제작비 계산, 카테고리 필터 및 배경색 구분
- 상품 가격 테이블: 정가/할인/할인가/할인가 순수익 계산
- 재료 재고 관리 테이블: 기본 재고 대비 판매 사용량/남은 재고 계산
- 예상 수익 시나리오: 상품별 예상 판매량 기반 총 예상 순수익 계산

모든 계산값은 소수점 3자리로 표시됩니다.
